# e-plantShopping

A React-based e-commerce shopping app built with Vite, Redux Toolkit, and React.

## Overview

**e-plantShopping** is an online plant store called **Paradise Nursery**.

## Features

- Browse plants by category
- Add plants to cart
- Increment and decrement quantities
- Remove items from cart
- View total cart amount

## Tech Stack

- React, Vite, Redux Toolkit
